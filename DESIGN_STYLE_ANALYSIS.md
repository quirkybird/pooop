# 项目设计风格分析

## 总体气质
- 关键词：温柔、私密、治愈、轻松、亲密互动
- 情绪表达靠色彩与文案：大量爱心/表情符号、暖色系与柔和绿让“健康记录”更亲近

## 颜色体系
- 主色：深草绿 `#184B22`，用于标题、边框、按钮主色
- 辅助色：嫩绿 `#7BD95B`、浅绿 `#B6E89B`，用于提示与高亮
- 暖色背景：奶油/米黄 `#FAF7E9`、`#F6F2E8`、`#FFF9E6`
- 情感强调：粉色系 `#FF6B9D`、`#FFE5EC`，用于伴侣卡片与互动按钮
- 交互强调：渐变黄色 `#FDE08B` → `#F9CB66` 作为主要 CTA
- 暗色模式：背景 `#121212`、`#1A1A1A`，但保持同一色系的柔和对比

## 排版与字体
- 字体对比：标题使用衬线 `Hedvig Letters Serif`，正文与 UI 用等宽 `JetBrains Mono`
- 这种“衬线 + 等宽”的组合带来“温柔 + 记录感”的气质
- 文本层级明确：大标题（3xl–7xl）强调情感，辅助说明用小号文本与较低不透明度

## 形状与空间
- 大量超大圆角（`rounded-2xl`、`rounded-3xl`、`rounded-[32px]`）
- 卡片式布局为主，内部间距充足，视觉非常“软”
- 阴影偏轻（`shadow-sm` / `shadow-lg`），营造柔和层次感

## 组件风格
- 导航：简洁、内容集中、强对比度文字、移动端下拉卡片
- 卡片：显式边框 + 柔和背景渐变（尤其是伴侣卡）
- 表单：多步流程，按钮与步骤条都有明显反馈
- 小提示：色块背景 + 小字号等宽字体，强调“温柔提醒”

## 动效与交互
- 强调“轻量反馈”：hover 放大、轻微上浮、阴影变化
- 过渡时长适中（`transition-all` / `duration-200`），避免突兀
- 动效主要用于按钮、卡片与表单步骤切换，增强愉悦感

## 图标与视觉符号
- 广泛使用 Lucide 图标 + Emoji
- Emoji 表达情绪，图标保证一致性与结构感

## 设计语言总结
- 这是一个“温柔的健康记录工具”
- 色彩与排版强调“亲密关系”和“轻反馈”
- 通过“圆角 + 柔和阴影 + 暖色背景 + 细腻渐变”建立疗愈感
- 强功能导向，但包装成轻松、关怀、低压力的体验
